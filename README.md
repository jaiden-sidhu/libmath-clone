# LibMath Clone

Clone of LibMath, a lightweight C++ math library providing utilities for arithmetic, matrix operations, prime number calculations, and more.

## Features
- Arithmetic functions 
- GCD functions
- Prime number functions
- Matrix functions
- Factorial functions
